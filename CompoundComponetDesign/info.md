### compound component design pattern

The compound component design pattern is a react design pattern that allows you to create reusable,
flexible and declarative components by composing multiple smaller components into a parent component.
These smaller components are designed to work together ,often sharing  the state and logic provided by the parent,
to build

- Work like parent child relationship
- Parent component will be provider
- Easy to maintain
- Parent component manage state and context acts like a controller
- Child component manage the presentation view of the application
